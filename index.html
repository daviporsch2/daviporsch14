O que é HTML?

📝 Definição

HTML (HyperText Markup Language) é a linguagem padrão para criar páginas web. Todos os sites que você visita são construídos com HTML.

🎯 Para que serve?

HTML estrutura o conteúdo da web: textos, imagens, vídeos, links, formulários e todos os elementos que vemos em um site.

⚡ Por que aprender?

É o ponto de partida para desenvolvimento web. Sem HTML, não existe site. É obrigatório para qualquer desenvolvedor.

🚀 Carreira

Conhecimento em HTML abre portas para Front-end, Back-end, Full Stack e muitas outras áreas da tecnologia.

Exemplo de Código

<!-- Estrutura básica de uma página HTML -->
<!DOCTYPE html>
<html>
    <head>
        <title>Minha Primeira Página</title>
    </head>
    <body>
        <h1>Olá, Mundo!</h1>
        <p>Este é meu primeiro site em HTML.</p>
        <a href="https://www.google.com">
            Clique aqui para buscar
        </a>
    </body>
</html>
